# Exam Day
A basic visual novel to learn Godot and Dialogic. 

## Planned Features
There are a few features that I hope to implement:
- Main menu
- Saving and loading
- Dialogue management
- Branching storylines
- In-game HUD for story purposes
- Music
- Sound effects
- Settings
